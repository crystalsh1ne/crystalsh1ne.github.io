# Ласкаво прошу
Зліва Ви зможете вибрати необхідну тему на її вивчення.\
Безпосередньо зправа від панелі ви зможете прочитати зміст теми.\
Текст, що позначається **жирним шрифтом** є обовʼязковим для вивчення (якщо Ви, звісно, хочете отримати високий бал на НМТ). Цебто дати, терміни або важливі поняття.
